# 🧩 TheABLEWay System (Tablet MVP)

> **Programmatic State Machine & Cognitive Scaffolding for Human Teams Under Stress.**

TheABLEWay is a bilingual (Greek/English), context-aware behavioral protocol built to stabilize human teams operating inside volatile, uncertain, complex, and ambiguous (VUCA) environments. Developed over 12 years of classroom testing and independent systems engineering, this application serves as an interactive tactical mirror that interrupts cognitive entropy and panic loops in under two minutes.

---

## 💎 Institutional & Scientific Validation

This framework operates at the precise interface of advanced cognitive science and operational architecture, utilizing two primary behavioral engines:

1. **4Es Cognition Model (Embodied, Embedded, Enacted, Extended):** The system serves as an *extended cognitive scaffold*, externalizing the user's executive function during periods of high amygdala hijacking and allostatic overload.
2. **Thaler’s Choice Architecture (Nudge Theory):** By embedding sequential boundary constraints directly into the workspace layout, the application enforces systematic reflection, breaking reactive behavioral patterns without top-down managerial coercion.

---

## 🕹️ Core Feature Architecture

* **Dual-Language Runtime:** Full localization scalability supporting `ΕΛΛΗΝΙΚΑ 🇬🇷` and `ENGLISH 🇺🇸` dynamically.
* **Context-Aware Adaptive Routing:** Distinct semantic prompts optimized for two primary user archetypes:
  * 💼 **Business / Corporate:** Focused on organizational roles, operational milestones, and resource locks.
  * 🏡 **Everyday Life / Relations:** Specialized sub-tracks targeting relationship friction (Love, Family, Finance), mapping systemic blindspots using conversational language.
* **Deterministic State Machine:** Built entirely on top of Streamlit session-state trackers, preventing data pollution and forcing compliance across sequential operational states.
* **Programmatic Workflow Freeze Engine:** A hard constraint circuit breaker applied to the **C3 (Connect/Mistakes)** node. If an agent attempts to bypass failure analysis to jump straight to execution (C4), the system locks the user interface, flashes an active warning box, and logs a friction event.
* **Anonymized Real-Time Telemetry Data:** Every user trajectory logs comprehensive interaction strings, processing speeds, anxiety-reduction indices, and freeze events into a production-ready payload structure.

## 📈 The 4 Cs Behavioral Trajectory (Phase 1: Discover)

The system maps organizational workflows against strict sequential action vectors:

* **C1: CHOOSE** -> Pinpoint active nodes/agents (Who is involved).
* **C2: COLLECT** -> Articulate isolated systemic needs (What is required).
* **C3: CONNECT** -> CRITICAL CONSTRAINT NODE: Map localized error parameters (Why it failed). Empty input triggers active WORKFLOW FREEZE.
* **C4: CREATE** -> Generate explicit, low-risk micro-actions (How to execute).

## 📦 Runtime & Deployment Specifications

### Prerequisites
The application runs entirely on a zero-cost infrastructure model using standard Python data libraries:
* `python >= 3.9`
* `streamlit`
* `pandas`

### Local Execution Instructions
To test the behavioral simulator locally on your machine or deploy it to a dedicated mobile/tablet browser frame:

1. Clone the repository:
   ```bash
   git clone https://github.com
   cd theableway-tablet
   ```
2. Install the lightweight runtime dependencies:
   ```bash
   pip install streamlit pandas
   ```
3. Run the Streamlit application layer:
   ```bash
   streamlit run app.py
   ```
